# La-cuenta-plis
Respositorio general para proyecto, donde se almacenan back y front end.

Para levantar el servidor se puede utilizar Visual Studio, para esto hay que hacer lo siguiente:
- Abrir la carpeta del proyecto y entrar a la carpeta "base_prueba" 
_ Abrir la carpet "login-base-2". 
- En la carpet "login-base-2" se encuentra el archivo llamado "server.js", hay que abrirlo.
- Apretar F5 o fn + F5.


Para el front se utiliza expo. Lo que hay que hacer es lo siguiente:
- Abrir la terminal del computador, como "cmd" en windows, y abrir el path hasta llegar a vistas.
- En esta carpeta correr el comando "expo start"
- Luego de esto se "abre" el programa en la misma terminal mostrando un código qr y distintas opciones de cómo se quiere mostrar la app. También se abre una página web con una UI por si no se quiere trabajar desde la terminal.
- Para abrirla en el celular hay que tener instalada la aplicación llamada "Expo", la cual se puede encontrar en appstore.
- Conectar el celular al computador en modo "debugger" y, o leer el código qr que aparece o escoger la opción donde se aprieta la letra "a", que es la que corresponde a android.
- Luego la app se empieza a cargar en el celular luego de unos segundos. Esta información o algún error, se puede encontrar en la pág. web con la UI mencionada anteriormente.
* Hay que tener instalado node.js
*https://expo.io/learn
